exception Halt of Rock.Response.t

The Halt exception can be raised to interrupt the normal processing flow of a request.

The exception will be handled by the main run function (in Server_connection.run) and the response will be sent to the client directly.

This is especially useful when you want to make sure that no other middleware will be able to modify the response.

val halt : Rock.Response.t -> 'a

Raises a Halt exception to interrupt the processing of the connection and trigger an early response.
